# Geopolitical Intelligence Automation System

A British-English portfolio repository for designing a semi-automated geopolitical intelligence workflow that turns information sources into reviewed intelligence records, historical analogue retrieval, evidence synthesis, analyst briefs, and dashboard-ready outputs.

## Purpose

This repository demonstrates:

- AI workflow design
- automation architecture
- intelligence operations
- dataset management
- decision-support thinking

It is not a trading system, forecasting engine, or investment recommendation tool.

## Core Workflow

```text
Information Sources
        |
        v
Candidate Event Queue
        |
        v
Human Review
        |
        v
Approved Intelligence Records
        |
        v
Historical Analogue Search
        |
        v
Evidence Synthesis
        |
        v
Analyst Brief
        |
        v
Dashboard
```

## Repository Structure

- `data/`: schema files for sources, candidate events, approved records, and analyst briefs.
- `docs/`: architecture, lifecycle, roadmap, and portfolio positioning documents.
- `scripts/`: future automation and validation scripts.
- `results/`: generated outputs from future pipeline runs.
- `dashboard/`: future static dashboard assets.

## Current Status

Phase 1 through Phase 3 are scaffolded as a system design foundation. The repository currently contains schemas and documentation only. It does not scrape websites, call LLM APIs, forecast outcomes, or provide investment advice.
